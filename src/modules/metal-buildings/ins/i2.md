# HUGE IMPORTANT THING

# You Need Boolean/Openings

Right now your wall probably:

* still exists behind the door

That destroys realism.

You need:

* wall cutouts/openings

Meaning:

```txt
wall geometry
MINUS
door opening
```

This is CRITICAL.

---

# Best Strategy

DO NOT use real-time boolean geometry yet.

Too expensive and buggy.

Instead:

# Procedural Wall Segmentation

Example:

Instead of:

```txt
Entire wall mesh
```

Generate:

```txt
left segment
door gap
right segment
top segment
```

This is how many configurators optimize openings.

Far cleaner.
