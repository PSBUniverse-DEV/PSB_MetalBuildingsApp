# Your Door System

This part is important:

You do NOT make:

* front door model
* back door model
* left door model
* right door model

That becomes duplication hell.

Instead:

```jsx
<RollupDoor
    wall="front"
    position={12}
    width={8}
    height={8}
/>
```

Then internally:

```js
switch(wall){
   case 'front':
   case 'back':
      rotateY(0)
      break;

   case 'left':
   case 'right':
      rotateY(Math.PI / 2)
      break;
}
```

ONE reusable component.

Not 4 copies.
